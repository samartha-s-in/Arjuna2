## Quick Installation

Run this single command on your Jetson Orin Nano Super:
```bash
bash <(curl -fsSL https://raw.githubusercontent.com/samartha-s-in/Arjuna2/main/Arjuna2_Setup.sh)
```

**What this does:**
- Downloads the setup script directly from GitHub
- Executes it immediately
- No manual download required
- Always uses latest version

**Requirements:**
- Jetson Orin Nano Super with JetPack 6.x
- Ubuntu 22.04
- Internet connection
- ~20GB free disk space
